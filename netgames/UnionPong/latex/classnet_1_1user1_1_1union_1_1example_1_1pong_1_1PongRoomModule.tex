\subsection{\-Detailed \-Description}
\-This is the \-Room\-Module that controls the pong game. \-The \-Reactor (\-Flash) client requests that the module be attached when it when it invokes room\-Manager()'s create\-Room() method.

\-The server will create a new instance of this module for each room.
